Projects

Beer Can Detection and Identification

This project required Object Detection and further classification of the beer cans of interest, while ignoring the others. Dataset had challenges like reflection of light from the ambient background on the glass-doors, due to which, a StyleGAN2 model was trained on web-scraped images with data manipulation. This made the GAN generate images that replicate the real world scenarios coarsely. This made our classification model using ResNet101 architecture more robust. Read More.


Multi-Modal Super Resolution for Dense Microscopic Particle Size Estimation

This project was used in an R&D pipeline, which involved accurate sizing of microscopic particles. Optical Microscope images were given as input. Super Resolved images that look like SEM were generated by a combination of AASNA-Pix2Pix and an Improved ESRGAN using Spectral Normalization. Further, a custom object detection model was created for the super resolved images: Super Resolution Particle Sizing Algorithm. The detection model gives output feature maps as 4D vectors, each pixel giving a map of (p, x, y, r) to detect the circle of a particle and its radius. We wrote a paper on this project which is currently awaiting approval by the legal teams.Read More.
